Terminal Commands

General

ctrl + LClear screen (previous output scrollable)

ctrl + _Undo the last change 🛈

ctrl + cTerminate/kill the current command or process

ctrl + zSuspend/pause the current process

ctrl + dLogout / close terminal, send EOF

ctrl + sFreeze terminal output 🛈

ctrl + qResume terminal output 🛈

ctrl + rReverse search the command history (repeat command to cycle)

↑/↓ Arrows Cycle command history

!!Run the last command

![command]Run the last command starting with character(s) OR line number 🛈

alt + .Insert the last argument from previous commands (repeat command to cycle)

tab Autocomplete commands, file & directory names

tab tabDisplays list of all possible completions when multiple/unknown options

ctrl+shift+cCopy to clipboard

ctrl+shift+vPaste from clipboard

Vi Commands

(from VIM Tutor)

Special Note:Many commands can be multiplied. For example, the dd command will delete a line. Entering d5d will delete 5 lines.

hMove cursor left

kMove cursor up

lMove cursor right

jMove cursor down

$Move cursor to end of the line

Esc keyNormal Mode

:q!Exits vi editor without saving changes

xDeletes character under the cursor

iInsert text mode (Escape key to exit insert mode)

AAppend text to end of line

:wqSave the file and exit

shift + zzSave the file and exit

dwDelete word from the cursor position until the start of the next word, EXCLUDING its first character 🛈

deDeletes everything from the cursor to the end of the current word, INCLUDING the last character

d$Deletes everything from the cursor to the end of the line, INCLUDING the last character

<num> wMove cursor a "number" of words forward EXAMPLE2w moves the cursor forward 2 words

<num> eMove cursor to the end a "number" of words forward EXAMPLE3e moves the cursor to the end of the third word forward

0 (number zero)Move cursor to the start of the line

d <num> wDeletes a "number" of words EXAMPLEd2w deletes 2 words

ddDeletes the whole line - Also functions like "cut"

uUndo the last command(s)

UUndo an entire line of command(s)

ctrl + rRedo the command(s) - Undo an undo

pPaste text on the next line after the cursor

r <char>Replace the character under the cursor EXAMPLErx would replace the character with the x character

ceDeletes from cursor to end of word and enters INSERT Mode

ccDeletes the entire line and enters INSERT Mode

c$Deletes from cursor to end of line and enters INSERT Mode

ctrl + gShow your location in file

shift + gMove cursor to end of file

ggMove cursor to start of file

<num> shift + gGo to line number you entered

/, n, N, ?Search functions are the same as man pages

ctrl + oGo back to previous position (letter O not zero)

ctrl + oGo forward to previous position

%When cursor is over a parentheses or bracket, it will move the cursor to the matching enclosing glyph -- *Useful in debugging a program with unmatched parentheses!

:s/new/oldReplace the first occurrence of the "old" word with "new" word in the line

:s/new/old/gReplace all"old" words with "new" word in the line

:#,#s/new/old/gReplace all "old" words with "new" word between the specified lines

:%s/new/old/gReplace all "old" words with "new" word in the file

:%s/new/old/gcReplace all "old" words with "new" word in the file but ask for confirmation each time

:! <command>Execute any terminal command from vi EXAMPLE::!ls (no space) -- Pressing ENTER afterwards returns to vi

:w <file_name>Saves the file under specified name

vStarts Visual Selection aka Highlighting text. Move cursor to highlight text. You can then perform a task on the highlighted text such as deletion or saving a portion of the text in a new file

:r <file_name>Insert contents from another file

:r !<command>Insert the output of an external command EXAMPLE::r !date

oStart a line below cursor in Insert mode (letter O not zero)

shift + oStart a line above cursor in Insert mode (capital O not zero)

aAppend text after the cursor

NOTE: a, i and A all go to the same Insert mode, the only difference is where the characters are inserted.

shift + rReplace mode starting at cursor position. Add new characters by typing over old characters

ycopy

ppaste

:set icIgnore case when searching

:set isShow partial matches for search phrase

:set hlsHighlight all matching search phrases

:set no<option>Turn off an option EXAMPLE::set nohls

:helpHelp menu

:help <option>Help for specific option

ctrl + wToggle help windows

:qQuit help

Command line completion with CTRL-D and <TAB> Type the start of a command:    :e
Press CTRL-D and Vim will show a list of commands that start with "e"
Type d<TAB> and Vim will complete the command name to ":edit"
