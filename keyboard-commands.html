Terminal Commands

Cursor

Positioning

ctrl + aStart of line

ctrl + eEnd of line

alt + bBack one word

alt + fForward one word

Functions

ctrl + uDelete everything left of cursor

ctrl + kDelete everything right of cursor

ctrl + wDelete word left of cursor

alt + dDelete word right of cursor

crtl + tSwap last 2 characters left of cursor

** Credit: Justin Lewis

Terminal Commands

General

ctrl + LClear screen (previous output scrollable)

ctrl + _Undo the last change 🛈

ctrl + cTerminate/kill the current command or process

ctrl + zSuspend/pause the current process

ctrl + dLogout / close terminal, send EOF

ctrl + sFreeze terminal output 🛈

ctrl + qResume terminal output 🛈

ctrl + rReverse search the command history (repeat command to cycle)

ctrl + rReverse search the command history (repeat command to cycle)

↑/↓ Arrows Cycle command history

!!Run the last command

![command]Run the last command starting with [characters inside brackets]

alt + .Insert the last argument from previous commands (repeat command to cycle)

tab Autocomplete commands, file & directory names

tab tabDisplays list of all possible completions when multiple/unknown options

ctrl+shift+cCopy to clipboard

ctrl+shift+vPaste from clipboard

** Credit: Justin Lewis
